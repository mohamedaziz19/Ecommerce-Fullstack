A full-stack e-commerce web application with a complete frontend and backend system.

## Tech Stack

### Frontend

- React (or Angular)
- Axios
- Bootstrap / Tailwind

### Backend

- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication

---

## Project Structure

client/ -> Frontend application
server/ -> Backend API

---

## Installation & Setup

### 1. Clone the repository

git clone https://github.com/your-username/ecommerce-fullstack.git
cd ecommerce-fullstack

---

### 2. Setup Backend

cd server
npm install
npm run dev # or npm start

---

### 3. Setup Frontend

cd client
npm install
npm start

---

## API Configuration

Create a `.env` file inside the client folder:

REACT_APP_API_URL=http://localhost:5000
